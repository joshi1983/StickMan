property AutoUpdate: Boolean

This is initially true.
When this is true, the leg's animation is updated automatically when the leg is drawn.

property Colour: tColor

The leg's colour

procedure DrawLeg(can: tCanvas;x,y: integer)

Draws the leg onto a canvas "can"

Parameters
can: tCanvas the canvas the leg is to be drawn on
x: integer the x coordinate of the hip joint
y: integer the y coordinate of the hip joint

procedure Kick(KickingStyle: tKickStyle;KickRight: boolean)

Starts a kicking animation
If the leg is already in a kicking motion, it ignores the request for another kick.

Parameters
KickingStyle: tKickingStyle The style of the kick. This could be ksReallySlowKick, ksSlowKick, ksQuickKick, or ksFlashKick
KickRight: Boolean This is true only when the leg is kicked to the right.

This sets importent initial properties for the leg.

Parameters
ang: real The angle of the leg
size: integer The length of the leg
VColour: tColor The colour of the leg

property Length: integer

The length of the leg

procedure Update

This is automatically called when the leg is drawn unless AutoUpdate is false.
