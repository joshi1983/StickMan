property AutoUpdate: Boolean

This is initially true.
When this is true, the Eye's animation is updated automatically when it is drawn

procedure Blink

This starts a blinking animation.
You can also call this while the eye is blinking.

property Blinking: Boolean

This is true only when the eye is in the process of blinking.
This is read only.

procedure DrawEye(can: tcanvas;x,y: integer)

This draws the eye on a canvas "can".

Parameters
can: tCanvas the canvas the leg is to be drawn on
x: integer the x coordinate of the centre of the eye
y: integer the y coordinate of the centre of the eye

property EyeColour: tColor

The colour of the eye

property Focus: tPoint

A point for the eye to track
You could use this to make is look like the eye is looking at the cursor.

procedure Initialize(sz: integer;EColour: tcolor)

This sets some importent initial properties of the eye

Parameters
sz: integer the size of the eye
EColour: integer the colour of the eye

procedure Update

This updates the eye's animation.
